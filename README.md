# MEAN-starter

Full stack web server based on node.js-Angular5-Express-MongoDB

## Installation
```prompt
$ git clone xxxx your-app-name
$ cd your-app-name
$ npm i
```

## Dev
Start node.js server with
```prompt
$ nodemon
````

Start angular server with
```prompt
$ ng serve
```

## Angular client
The angular client contents the following assets:
- material.angular
- routing file
